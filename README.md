# nonebot2
demo机器人，不用再担心自己是小白！
小白专用，避免各种配置出现问题，你只需要换上自己扣扣账号和密码就行了，相关配置已经配好，已发布几个可使用插件，建议用小号哦，亲！喜欢就点个star吧。当然，你也可以账号密码留空，手机扫码登陆即可。注意关闭qq登录保护。
